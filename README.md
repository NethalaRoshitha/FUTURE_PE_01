# AI Website Copy Generator – Prompt Engineering Task

## 📌 Business Chosen
Glow Beauty Salon, Vijayawada

## 🎯 Objective
To design a structured prompt system that generates high-quality, conversion-focused website content for local businesses using AI.

## 🛠️ Tools Used
- ChatGPT (for content generation)
- Lovable AI (for website creation)
- GitHub (for documentation and submission)

## ✨ What I Built
- Homepage copy (headline, subheadline, introduction)
- Services page content (Hair, Skin, Makeup)
- Call-to-action (CTA) sections
- Website preview based on generated content

## 🧠 Prompt Strategy
I used structured, role-based prompts by defining:
- Business context (local salon)
- Target audience (modern women)
- Tone (friendly, premium, persuasive)
- Clear output sections (homepage, services, CTA)

This approach helped generate specific, high-quality, and non-generic content suitable for real-world use.

## 🌐 Website Preview
A website was generated using Lovable AI based on the AI-generated content.
